### 🤖 [SocialCraft AI](https://github.com/shre111/socialcraft-ai) — Multilingual AI Caption Generator
Multilingual social media caption generator with self-learning AI personalization. Built with Next.js 14 + FastAPI + Claude + Supabase.

### 🎙️ [Seed-VC Pipeline](https://github.com/shre111/seed-vc-pipeline) — AI Voice Cloning + Talking Avatar
End-to-end pipeline for AI voice cloning and talking avatar video generation using Seed-VC + SadTalker + Node.js + React.

Full-stack lead management and CRM application with live deployment on Vercel. Built in TypeScript.
